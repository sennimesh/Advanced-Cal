Advanced Calculator

An advanced web-based calculator with scientific functions built using **HTML, CSS, and JavaScript**.  
This project is designed to demonstrate logical problem-solving, UI design, and JavaScript functionality.

Features

- Basic arithmetic operations (+, −, ×, ÷)
- Scientific functions:
  - sin, cos, tan
  - log, ln
  - square root
- Brackets support `( )`
- Clear (C) and Delete (DEL) buttons
- Responsive and user-friendly interface
- Error handling for invalid expressions

Technologies Used

- HTML5 – Structure
- CSS3 – Styling & layout
- JavaScript (ES6) – Calculator logic
